Web Proxy

A Web Proxy is a local server, which fetches items from the Web on behalf of a Web client instead of the client fetching them directly. This allows for caching of pages and access control.

Built in Java, the WebProxy class contains the implementation for the proxy itself & HTTP caching. The ManagementConsole class contains the dynamic URL blocking functionality and displays incoming requests. Repo is here.
